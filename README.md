💰 Finance Dashboard

A modern, responsive Finance Dashboard UI built with React that helps users track income, expenses, and financial insights with a clean and intuitive interface.

---

📌 Features

📊 Dashboard Overview

- Total Balance, Income, and Expenses summary cards
- Balance trend visualization (line chart)
- Spending breakdown (category chart)

---

📋 Transactions Management

- View all transactions in a structured table
- Search and filter transactions
- Add new transactions (Admin only)
- Edit existing transactions (Admin only)
- Delete transactions (Admin only)

---

🔐 Role-Based UI

- Viewer
  - Can only view data
- Admin
  - Can add, edit, and delete transactions

---

💡 Insights Section

- Total spending calculation
- Highest spending category
- Basic financial observations

---

💾 Data Persistence

- All transactions stored in localStorage
- Data remains after page refresh

---

🧠 Tech Stack

- Frontend: React (Vite)
- State Management: Zustand
- Styling: Tailwind CSS
- Charts: Recharts
- Notifications: React Hot Toast

⚙️ Installation & Setup

1. Clone the repository

2. Navigate to project

cd finance-dashboard

3. Install dependencies

npm install

4. Run development server

npm run dev
